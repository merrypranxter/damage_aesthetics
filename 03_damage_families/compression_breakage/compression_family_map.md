# COMPRESSION FAMILY MAPThis file maps the complete compression breakage family across digital media contexts—artifacts of insufficient or failed data reduction.## Core behaviorVisible degradation from mathematical approximation, quantization, and predictive encoding.## Family members### Macroblocking- **Character:** visible block grid, texture loss- **Cause:** coarse DCT quantization- **Visibility:** motion, gradients, low bitrate### Banding / posterization- **Character:** stepped gradients, tonal jumps- **Cause:** insufficient bit depth, quantization- **Visibility:** smooth tonal regions### Mosquito noise- **Character:** edge chatter, fine speckle- **Cause:** ringing + quantization around edges- **Visibility:** high-contrast boundaries### Ringing / edge echo- **Character:** overshoot/undershoot halos- **Cause:** Gibbs phenomenon, filter artifacts- **Visibility:** sharp edges, text### Temporal smear / prediction error- **Character:** motion trails, ghosting, melt- **Cause:** interframe prediction failure- **Visibility:** motion, scene changes### Datamosh residue- **Character:** block migration, shape borrowing- **Cause:** prediction abuse or corruption- **Visibility:** extreme temporal corruption### Bitrate starvation- **Character:** texture collapse, detail loss- **Cause:** insufficient bits for complexity- **Visibility:** fine detail, motion,
