# SEVERE FAILURE STACKSThis file documents extreme damage combinations—stacks where multiple severe artifacts create near-total information collapse.## DefinitionSevere failure stacks are combinations of high-severity (S4–S5) artifacts from multiple families, resulting in dramatic legibility loss and aesthetic transformation.## The Signal Death Stack### Components (all S4–S5)- **Broadcast sync collapse** — rolling, tearing, static takeover- **VHS tracking failure** — horizontal band chaos- **CRT convergence error** — color separation, geometry failure- **Analog noise field** — snow overwhelming content### ResultNear-total image collapse. Occasional recognizable forms emerge from noise. Strong "lost signal" aesthetic.### UseHorror liminality, analog horror, technological sublime---## The Digital Decay Stack### Components (all S4–S5)- **Severe macroblocking** — block grid dominates- **Datamosh residue** — temporal hallucination- **Packet loss fragmentation** — image shattering- **Recompression banding** — gradient destruction- **Screenshot generation loss** — accumulated decay### ResultComputational breakdown. Image becomes abstract block field. Time and space corrupted.### UseDigital horror, network anxiety, contemporary decay---## The Chemical Destruction Stack### Components (all S4–S5)- **Film vinegar syndrome** — base shrinkage, emulsion buckling- **Color dye total fade** — monochrome or shifted remnant- **Mold/biological damage** — organic patterning- **Water damage** — emulsion loss, staining- **Physical warping** — dimensional distortion### ResultMedium destruction. Image barely preserved on damaged substrate. Archaeological ruin.### UseMemory loss, archival tragedy, time's violence---## The Chain Collapse Stack### Components (cumulative severity)- **Original:** Film (heavy damage)- **Copy 1:** VHS (generation loss)- **Copy 2:** Digitized (early, poor scanner)- **Copy 3:** Compressed (aggressive codec)- **Copy 4:** Platform recompressed- **Copy 5:** Screen-recorded- **Copy 6:** Re-uploaded### ResultGeological layers of damage. Original barely discernible. Each generation adds signature.### UseInternet archaeology, meme decay, digital folklore---## Designing severe stacks### 1. Multiple familiesInclude artifacts from at least 3 different families.### 2. Severity consistencyAll components at S4–S5, or clear progression.### 3. Legibility controlKnow what remains readable and why.### 4. Affective targetHorror? Sublime? Tragedy? Absurdity?### 5. Medium coherenceEven severe stacks should have internal logic.## Common severe stack failures### Random severityMixing S1 and S5 without progression logic### Single-family overloadOne artifact type repeated rather than layered### Total abstractionNo recognizable content—becomes texture, not damaged image### Incoherent medium mixingContradictory artifacts without chain explanation## Prompting severe stacksBuild by escalation:> "severe VHS tracking failure with horizontal tearing, analog snow takeover, CRT convergence error causing color separation, image nearly lost to signal collapse"Each component adds to the breakdown.## SummarySevere failure stacks are the aesthetic of system collapse—where multiple failures converge to transform or destroy the image entirely.
