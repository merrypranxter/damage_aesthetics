# PAUSE, WARP, FREEZE, ECHOThis file documents specific playback state artifacts: what happens when playback pauses, warps, freezes, or repeats in analog and digital systems.## DefinitionThese are temporal artifacts caused by playback state changes:- **Pause**: still frame with possible degradation- **Warp**: speed distortion (slow/fast/slurring)- **Freeze**: stuck frame or stutter-loop- **Echo**: frame repetition or persistence## Core visual identity### Pause artifacts (analog)- degraded still image- head switching noise becoming static- chroma instability- potential burn-in on displays### Pause artifacts (digital)- clean freeze or blocky freeze- possible macroblocking in still- buffer state visible### Warp/slurring- pitch-shifted audio (if present)- temporal stretching/compression- motion smear or strobe- mechanical "eating" of tape### Freeze/echo- repeated frames- stutter loops- persistence of motion- buffering rhythm## Causes- user pause command- mechanical tape issues (analog)- buffer states (digital)- error concealment- network streaming adaptation- deliberate artistic effect## Severity rangeS1–S2: brief pause, subtle warpS3: obvious freeze, clear echo/repeatS4–S5: extended freeze, severe warp, broken playback## Confusion risks### Digital freeze vs analog pauseDigital freezes clean or blocky; analog degrades and noise-builds.### Echo vs ghost framesEcho is playback repetition; ghost is signal/memory persistence.## Stack partnersOften with:- native media artifacts- display artifacts- compression artifacts (digital)## Prompting notesUseful:- paused VHS degradation- digital freeze-frame with block artifacts- playback warp and slur- stuttering frame echo## SummaryThese are the artifacts of playback interrupted—time itself stuttering, stretching, or freezing as the machine struggles to continue.
