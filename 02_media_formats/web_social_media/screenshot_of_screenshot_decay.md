# SCREENSHOT OF SCREENSHOT DECAYThis file documents the specific degradation that occurs when images are repeatedly screen-captured, saved, and re-uploaded—creating a distinctive chain of recompression and resolution loss.## DefinitionScreenshot-of-screenshot decay refers to the accumulated damage from iterative screen capture, saving, and re-sharing, particularly on platforms that recompress or resize images.## Core visual identityThis decay feels:- progressively softer- contrast-flattened- recompression-artifact laden- resolution-reduced- platform-specific in its ugliness- historically layered like geological strata## Common visible traits### Generation 1- original quality, possibly with UI chrome### Generation 2–3- slight softness from scaling- first recompression artifacts- possible caption or UI addition### Generation 4–6- obvious JPEG blockiness- color shifts- edge degradation- text becoming fuzzy### Generation 7+- severe artifact buildup- near-illegibility of fine detail- "deep fried" aesthetic- color and contrast wildly shifted- almost abstract texture## Platform-specific signaturesDifferent platforms add different damage:- Twitter: aggressive JPEG, color shift- Instagram: multiple aspect crops, heavy compression- Facebook: notorious recompression- Messaging apps: variable quality, thumbnail generation## Causes- resolution reduction at each capture- JPEG recompression artifacts- color space conversion- platform-specific compression profiles- UI chrome accumulation- cropping and rescaling## Severity rangeMeasured by generation and platform toxicity.G1: S0–S1G2–G4: S2–S3G5+: S4–S5, "deep fried" territory## Confusion risks### Screenshot decay vs original low qualityDecay is layered; original low quality is single-generation.### Deep fried vs deliberate distortionDeep fried is accumulated; deliberate is applied.## Stack partnersInherently stacked with:- JPEG artifacts- platform recompression- possible caption/text overlays- UI elements## Prompting notesUseful:- 10th-generation screenshot decay- deep fried meme compression- accumulated Twitter recompression- iterative screenshot quality loss## SummaryScreenshot-of-screenshot decay is the geology of digital sharing—each layer of capture and compression burying the original under strata of platform-specific damage.
